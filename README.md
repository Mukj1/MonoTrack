### 中文使用说明

在 [Gemini 3](https://ai.google.dev/gemini-api/docs/gemini-3?hl=zh-cn&thinking=high) 的帮助下，MonoTrack 得以诞生。它允许你上传多个 FIT 或 GPX 运动轨迹文件，并将它们同时可视化在同一张地图上，帮助你对比每一次运动。

本网页程序提供中英文双语版本，您可以通过点击侧边栏的语言按钮切换语言。
同时，程序还提供了按运动类型和日期区间筛选轨迹的功能，帮助您更精确地查看和分析数据。


如需导出自己的全部运动轨迹以使用MonoTrack，请参考 [Strava 官方说明](https://support.strava.com/hc/en-us/articles/216918437-Exporting-your-Data-and-Bulk-Export)。



网页访问请前往：https://mono-track.vercel.app/

### User Guide (English)

With the help of [Gemini 3](https://ai.google.dev/gemini-api/docs/gemini-3?hl=zh-cn&thinking=high), MonoTrack was created. It allows you to upload multiple FIT or GPX activity files and visualize them on the same map, enabling you to compare each workout.

This web application is available in both English and Chinese. You can switch between the two languages by clicking the language buttons in the sidebar.
It also offers options to filter tracks by activity type and date range.

To export your activity data for use with MonoTrack, please refer to the Strava [Exporting your Data and Bulk Export](https://support.strava.com/hc/en-us/articles/216918437-Exporting-your-Data-and-Bulk-Export).


Access the web application here: https://mono-track.vercel.app/
