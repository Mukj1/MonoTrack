
在 [Gemini 3](https://ai.google.dev/gemini-api/docs/gemini-3?hl=zh-cn&thinking=high) 的帮助下，MonoTrack 得以诞生。它允许你上传多个 FIT 或 GPX 运动轨迹文件，并将它们同时可视化在同一张地图上，帮助你对比每一次运动。

如需导出自己的全部运动轨迹，请参考 Strava 官方说明：https://support.strava.com/hc/en-us/articles/216918437-Exporting-your-Data-and-Bulk-Export

网页访问请前往：https://mono-track.vercel.app/
